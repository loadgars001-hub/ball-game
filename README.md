# ball-game
This is a game about a ball that you have to push with your cursor to the end of the level. Along the way, you will encounter a yellow circle.
There will be obstacles such as walls, moving platforms, and Boost objects that push the player.
There is a level editor where you can create your own levels.
The game has a Lua scripting system that allows you to create custom item logic.
These items are available:

* Coin
* Trap
* Boost
* SpinBoard
* Blank

Each item has a basic Lua script that you can expand or modify.
(Scrolling is not implemented yet, but it will be added in future versions.)
Local multiplayer is implemented, allowing you to play with others over a local network.
Ball Color currently has 4 available colors.
